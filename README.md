# 💼 Salary Prediction using Linear Regression

## 📌 Project Overview

This project aims to predict employee salary based on factors like age, experience, gender, and education level using Machine Learning.

---

## 📊 Dataset Features

* Age
* Years of Experience
* Gender
* Education Level
* Salary (Target)

---

## 🔍 Steps Performed

### 1. Data Cleaning

* Handled missing values
* Removed unnecessary columns (Job Title)

### 2. Exploratory Data Analysis (EDA)

* Correlation heatmap
* Boxplots for categorical variables

### 3. Feature Engineering

* Gender encoding
* One-hot encoding for education level

### 4. Model Building

* Linear Regression model
* Train-test split (80-20)

### 5. Feature Scaling

* Applied StandardScaler to numerical features

### 6. Model Evaluation

* R² Score
* Adjusted R² Score

---

## 📈 Results

* R² Score: ~0.90
* Adjusted R²: ~0.89

👉 Model shows strong predictive performance with minimal overfitting.

---

## 🧠 Key Learnings

* Importance of avoiding data leakage
* Proper ML pipeline design
* Feature scaling and encoding
* Model evaluation techniques
